Скругление уголков (border-radius)

Обычное скругление

border-radius: 10px

Круг

border-radius: 50%

Эллипс

border-radius: 75px / 50px

Диагональное скругление

border-radius: 20px 50px

Разное для каждого угла

border-radius: 15px 30px 45px 60px

Сложная форма

border-radius: 100% 0% 100% 0% / 0% 100% 0% 100%

Синтаксис border-radius:
• Одно значение: все углы одинаковые
• Два значения: top-left+bottom-right / top-right+bottom-left
• Четыре значения: top-left / top-right / bottom-right / bottom-left
• Эллипс: горизонтальный_радиус / вертикальный_радиус
/* Круг */
border-radius: 50%;

/* Эллипс */
border-radius: 75px / 50px;

/* Разные углы */
border-radius: 15px 30px 45px 60px;

/* Сложная форма */
border-radius: 100% 0% 100% 0% / 0% 100% 0% 100%;
