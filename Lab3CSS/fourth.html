Задание №6: Свойства max-width и min-width

max-width: 300px

Элемент с max-width: 300px. Ширина ограничена сверху.

min-width: 300px

Элемент с min-width: 300px. Ширина ограничена снизу.

Сравнение поведения:

/* max-width ограничивает максимальную ширину элемента */ .element { max-width: 300px; } /* min-width устанавливает минимальную ширину элемента */ .element { min-width: 300px; }

Наблюдение: max-width позволяет элементу быть уже установленного значения, но не шире. min-width, наоборот, позволяет элементу быть шире, но не уже установленного значения.

/* max-height ограничивает максимальную высоту элемента */ .element { max-height: 70px; } /* min-height устанавливает минимальную высоту элемента */ .element { min-height: 80px; }

Наблюдение: max-height ограничивает высоту элемента сверху, даже если задано большее значение height. min-height гарантирует, что элемент будет не ниже установленного значения, даже если контента мало.

Практическое применение всех свойств вместе

Этот элемент имеет:
  • width: 80% (относительная ширина)
  • min-width: 200px (не может быть уже)
  • max-width: 600px (не может быть шире)
  • height: 150px (фиксированная высота)
  • min-height: 100px (не может быть ниже)
  • max-height: 200px (не может быть выше)
