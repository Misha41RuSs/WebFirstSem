Задание №1: Псевдокласс :focus
Псевдокласс :focus применяется к элементу, когда он получает фокус (например, при клике или переходе по Tab). Часто используется для улучшения доступности и UX форм.
.focus-input:focus {
  border-color: #3498db;
  box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.3);
  background-color: #ecf0f1;
  transform: translateY(-2px);
}

::before создает псевдоэлемент, который является первым потомком выбранного элемента. Часто используется для добавления декоративного контента перед содержимым элемента.
li::before {
  content: "★";
  color: #e74c3c;
  margin-right: 15px;
  font-weight: bold;
}
Задание №4: Псевдоклассы ссылок

Альтернативный стиль

Псевдоклассы ссылок:
• :link - непосещенная ссылка
• :visited - посещенная ссылка
• :hover - при наведении курсора
• :active - в момент активации (клика)
a:link { color: #3498db; }
a:visited { color: #8e44ad; }
a:hover {
  background: #3498db;
  color: white;
  transform: translateY(-3px);
}
a:active { background: #2980b9; }
Задание №5: Псевдоэлементы ::first-letter и ::first-line

Это первый абзац текста, который демонстрирует использование псевдоэлементов. Первая буква должна быть зеленого цвета и размером 24pt, а первая строка - синего цвета. Весь остальной текст абзаца отображается красным цветом с размером 12pt, как того требует задание.

Второй абзац также следует тем же правилам форматирования. Здесь мы видим, как псевдоэлементы могут быть использованы для создания визуально привлекательных текстовых блоков без изменения разметки HTML. Это мощный инструмент для веб-дизайнеров.

Третий пример абзаца показывает согласованное применение стилей ко всем параграфам. Такой подход обеспечивает единообразие оформления и экономит время разработки, поскольку не требует ручного форматирования каждого элемента в отдельности.

::first-letter выбирает первую букву текста элемента.
::first-line выбирает первую строку текста элемента.
p {
  color: red;
  font-size: 12pt;
}

p::first-letter {
  color: #2ecc71;
  font-size: 24pt;
  font-weight: bold;
}

p::first-line {
  color: #3498db;
  font-weight: bold;
}
Задание №6: Различные псевдоэлементы

Элемент с ::before и ::after

Этот блок демонстрирует использование псевдоэлементов ::before и ::after для добавления декоративного контента в начало и конец элемента.

Выделение текста ::selection

Попробуйте выделить этот текст мышью. Псевдоэлемент ::selection позволяет стилизовать выделенный пользователем текст.

Плейсхолдер input::placeholder

Другие полезные псевдоэлементы:
• ::after - создает элемент после содержимого
• ::selection - стилизует выделенный текст
• ::placeholder - стилизует текст-подсказку в полях ввода
.element-box::before {
  content: "«";
  color: #e74c3c;
  font-size: 40px;
}

.highlight::selection {
  background: #f39c12;
  color: white;
}

input::placeholder {
  color: #e74c3c;
  font-style: italic;
}
