Демонстрация всех типов позиционирования

Типы позиционирования в CSS:

Static (статическое) - позиционирование по умолчанию
Relative (относительное) - смещено относительно нормального положения
Absolute (абсолютное) - позиционируется относительно ближайшего позиционированного предка
Sticky (липкое) - переключается между relative и fixed в зависимости от прокрутки
Fixed (фиксированное) - позиционируется относительно окна браузера

Ключевые различия:

  • Static: Элемент располагается в нормальном потоке документа
  • Relative: Смещается относительно своего нормального положения
  • Absolute: Удаляется из нормального потока, позиционируется относительно предка
  • Fixed: Фиксируется относительно окна просмотра
  • Sticky: Ведет себя как relative до определенной точки прокрутки, затем как fixed
